Описание проекта для GitHub

Это Django-приложение для получения солнечной активности, API к данной программе написаны самостоятельно, но можно установить и любые другие.

## Основные функции

- Получение солнечной активности в определенную дату.
- Данные получаются со сторонних API, предусмотрено два API в случае неработоспособности одного из них, будут приходить данные со второго и наоборот.
- В случае неработоспособности обоих API будет выводиться сообщение о том, что сервис сейчас не доступен.
